# Personal Website — Agastya Iyengar

Personal portfolio site built with React, Tailwind CSS, and vanilla JSX (no build step).

## Structure

- **Personal tab** — about me, pursuits, and a photo from Florence
- **Professional tab** — skills, overview, and selected projects

## Running locally

```bash
cd scripts
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080).

## Stack

- React (via CDN)
- Tailwind CSS (via CDN)
- Custom `<image-slot>` web component for drag-and-drop photos

## Deploying

Push to `main` and the site is live via GitHub Pages (if enabled) or any static host.
